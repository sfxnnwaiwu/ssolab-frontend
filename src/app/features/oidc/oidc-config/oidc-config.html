Configuration Tips

  • • All fields are required for OIDC authentication
  • • The discovery URL auto-configures endpoints from your IdP
  • • Authorization Code flow is the most secure option
  • • The "openid" scope is mandatory and cannot be deselected
  • • Use "Test Connection" to validate before testing authentication
